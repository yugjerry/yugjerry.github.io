How to have fonts embedded in your pdflatex file? (Use Ghostscript)

I especially deal with pdf figures exported from R/Python. 

gs -dNOPAUSE -dBATCH -sDEVICE=pdfwrite -dPDFSETTINGS=/prepress -dEmbedAllFonts=true -sOutputFile=[your new filename] -f [your original filename]

S&P500 stocks data for time series analysis: yfinance in python

import yfinance as yf
import numpy as np

start_date = "2015-01-01"
end_data = "2019-12-31"

stock_names=np.genfromtxt("stock_names.csv",delimiter="\n",dtype=str)
space = ' '
stock_code = space.join(stock_names[0:200])
data = yf.download(stock_code, start=start_date, end=end_data)
